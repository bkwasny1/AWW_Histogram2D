W tym folderze umieszczamy zdjęcia (obsługiwane formaty: .tif, .bmp) o nazwie w postaci:  "input<numer>.<rozszerzenie>"
Zdjęcia z tego folderu aplikacja wykorzystuje do uruchomienia algorytmu w pętli nieskończonej jak również w określonej liczbie iteracji.

!!! numeracje obrazów należy rozpocząć się od 0 !!!
